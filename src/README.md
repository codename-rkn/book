# Introduction

## Description

Apex Recon is a Web Application Attack Surface Mapper, capable of analyzing the behavior 
and I/O characteristics of modern web applications and web APIs.

You can access Apex Recon via multiple interfaces, such as:

* [CLI](./interfaces/cli.md)
* [MCP API](./interfaces/mcp/index.md)
* [Web UI](./interfaces/web/index.md)
